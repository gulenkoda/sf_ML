## Описание проекта

Прототип системы компьютерного зрения для автоматического мониторинга зон барбекю и мангалов. 
Система определяет наличие открытого огня и людей рядом с ним, помогая предотвращать пожары 
и контролировать соблюдение правил безопасности.

**Ключевая задача:** Обнаружение потенциально опасных ситуаций, когда люди находятся вблизи открытого огня.

## Решаемая бизнес-задача
Автоматизация мониторинга зон отдыха с мангалами для:
- Предотвращения лесных пожаров
- Контроля соблюдения правил пожарной безопасности
- Снижения нагрузки на персонал служб контроля
- Раннего предупреждения опасных ситуаций


## Используемые технологии
- **Язык программирования:** Python 3.11+
- **ML-инфраструктура:** Roboflow Inference API
- **Библиотеки компьютерного зрения:** Supervision, OpenCV, PIL
- **Визуализация:** Matplotlib
- **Модель:** Предобученная модель детекции огня и людей
