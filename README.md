## Описание проекта

Прототип системы компьютерного зрения для автоматического мониторинга зон барбекю и мангалов. 
Система определяет наличие открытого огня и людей рядом с ним, помогая предотвращать пожары 
и контролировать соблюдение правил безопасности.

**Ключевая задача:** Обнаружение потенциально опасных ситуаций, когда люди находятся вблизи открытого огня.

## Решаемая бизнес-задача
Автоматизация мониторинга зон отдыха с мангалами для:
- Предотвращения лесных пожаров
- Контроля соблюдения правил пожарной безопасности
- Снижения нагрузки на персонал служб контроля
- Раннего предупреждения опасных ситуаций

## Возможности

- Обнаружение открытого огня
- Детекция людей
- Определение опасных ситуаций
- Визуализация с bounding boxes

## Используемые технологии
- **Язык программирования:** Python
- **ML-инфраструктура:** Roboflow Inference API
- **Библиотеки компьютерного зрения:** Supervision, OpenCV, PIL
- **Визуализация:** Matplotlib
- **Модель:** Предобученная модель детекции огня и людей

## Установка и запуск

### Требования
- Python не старше 3.12 или новее
- Доступ к интернету (для работы с Roboflow API)

### Установка зависимостей и библиотек
1) inference-sdk 
2) matplotlib
3) numpy
4) PIL

Используйте команду pip install в терминале VS Code

Также рекомендуется создавать собственный venv под этот проект

**Уточнение:** Если что-то не заработало, попробуйте:
pip install inference-sdk supervision pillow matplotlib requests numpy

## Краткая инструкция по работе с MVP

### Что делать?

1) Скачайте файл с программой на свое устройство
2) Установите нужные зависимости и библиотеки
3) В переменную IMAGE_DATA вставьте ссылку на изображение из интернета
4) Запустите по очереди все разделы сверху вниз
5) Наслаждайтесь результатом

### Примеры выводов

- ОПАСНОСТЬ! Обнаружен открытый огонь. Рядом с огнем 3 человека.
- Обнаружен открытый огонь. Людей рядом нет.
- Люди обнаружены: 2 человека. Открытого огня нет.
- Обнаружен открытый огонь, но люди находятся на безопасном расстоянии.
- Ничего не обнаружено: нет ни огня, ни людей.
